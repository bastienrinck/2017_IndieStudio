# 2017_IndieStudio

**Subject [here](https://github.com/thibautcornolti/IndieStudio/blob/master/Subject.pdf)**

Home made cross plateform "Bomberman" game, using irrlicht 3D Engine, irrklang audio library and a dedicated AI.
